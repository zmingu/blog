# 移动
- 基础h j k l
- 复制行(yank): yy
- 粘贴(paste): p

# 删除
- 删除(delete): d
- 删除单词并进入插入模式: ciw
- 内容选择模式: v
- 块级删除: di*, da*, ci*

# 插入
 - 在光标之前插入：i
 - 在本行开头插入：I
 - 在光标之后插入：a
 - 在本行结尾插入：A
 - 在本行之后新增一行插入：o
 - 在本行之前新增一行插入：O
 - 删除当前字符插入：s
 - 删除当前行中的所有文本插入：S
 

# 跳跃
- 后跳找字母：f*
- 前跳找字母：F*
- 跳行首：^
- 跳行尾：$
- 括号匹配跳转：%
- 跳到上一个字符的定位处：*
- 跳多次：[n]h/j/k/l
- 跳第一行：gg
- 跳最后一行：G
- 跳当前页的第一行：H
- 跳编辑器中部：M
- 跳到当前页的最后一行：L
- 跳到后面 word 的开始位置，前面加上数字 代表重复多少次：[n]w
- 跳到下一个词语的末尾：e
- 跳到上一个单词的开头：b

# 改变窗口的整体位置
- 将当前光标所在处 移动到窗口中部：zz
